零成本搭建个人博客全记录

关于本文：一个 50 岁的建筑人，用 AI 帮忙、GitHub 管理、Vercel 部署，零成本搞定自己的博客站。不是技术教程，是我的实操记录。

我想有一个自己的博客。

不是为了当博主、赚流量。是为了有一个地方，能把我维权的过程、学 AI 的笔记、创业的想法整理起来。不是为了给别人看——虽然有人看当然好——首先是为了给自己一个交代。

但我不会写代码。一个建筑设计师，会画图、会管项目、会用 CAD，但 HTML 和 CSS 对我来说是天书。

然后我发现了 AI 写代码这件事。

为什么不做公众号

朋友建议我开个公众号就行。我考虑过，但有几个原因让我决定还是搞个独立站：

  • 所有权——公众号的内容在微信的池子里，说封就封。自己的网站，内容在自己手上。
  • 自由度——想写什么写什么，没有审核、没有敏感词、没有流量算法的绑架。
  • 学习价值——建站本身就是一次 AI 实操。一个 50 岁的人，学学怎么用 AI 建网站，这件事本身就很有意思。
  • 成本为零——代码 AI 写，托管 GitHub 免费，部署 Vercel 免费。除了域名可能花几十块，其他全是零成本。

用到的工具

  • Claude Code——AI 编程助手，帮我写 HTML、CSS、JS
  • GitHub——免费代码托管，版本管理
  • Vercel——免费部署，自动从 GitHub 同步更新
  • VS Code——写代码的编辑器（其实大部分代码是 AI 写的，我只需要告诉它我要什么）

搭建过程（非技术版）

我不打算写技术教程——网上已经有太多更好的教程了。我只说说一个非技术人员做这件事的真实感受。

第一步：告诉 AI 我想要什么

我跟 Claude 说：帮我做一个个人网站，要有首页、关于页、博客列表，设计要简洁大方。它就开始生成了。

第一次看到它刷刷刷地写出几十行代码的时候，我是真的有点震惊。不是因为技术多厉害——而是因为我发现，我只需要说人话，它就能干活了。

第二步：不满意就改

设计风格不喜欢？改。布局不对？改。颜色太冷？换暖色。全部用自然语言跟 AI 沟通——"把蓝色改成深一点的颜色"、"卡片加个 hover 阴影效果"、"字体大一点"。

整个过程就像在跟一个很听话的设计师沟通。不用学 CSS、不用查文档，说出你想要的效果，它帮你实现。

第三步：一键部署

代码写好以后，推送到 GitHub，Vercel 自动部署。每次更新，推送上去等一两分钟就能看到效果。

从开始建站到上线，用了不到一天。而我一行的原生代码都没写。

一个建筑人的体会

做了二十年建筑设计，我发现建网站和做建筑在本质上是一样的：

  • 先有结构，再有装饰。——HTML 是框架，CSS 是外立面，JS 是设备系统
  • 迭代是最好的设计方式。——先上线、再优化。别等"完美"再发布
  • 工具不重要，想法重要。——以前是 CAD 和 BIM，现在是 AI。工具一直在变，但你要做什么，这个只有你自己知道

下一步

博客搭好了，接下来就是认真写内容。计划中的系列：

  • 维权日记——记录劳动仲裁的全过程
  • AI 学习笔记——一个中年人的 AI 入门记录
  • 创业思考——中年创业的想法和准备
  • 研究生生活——重返校园的真实体验

如果你也是这个年纪、也在学新东西、或者也在经历人生的转折——欢迎常来看看。咱们这个岁数的人，不服输。
