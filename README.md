# 老舟的 AI 探索笔记

一个 50 岁建筑人的 AI 学习记录。分享 AI 心得、AI 作品和 AI 工具资源。

## 技术栈

- 纯 HTML / CSS / JavaScript（第一版）
- 后期计划升级到 Astro

## 部署

1. 推送代码到 GitHub 仓库
2. 在 Vercel 中导入该仓库
3. 自动部署（Vercel 无需配置即可部署静态站点）

## 目录结构

```
/
├── index.html        # 首页
├── blog.html         # 博客列表
├── tools.html        # 工具导航
├── about.html        # 关于页
├── blog/             # 博客文章
├── css/
│   └── style.css     # 全局样式
└── js/
    └── main.js       # 交互逻辑
```
